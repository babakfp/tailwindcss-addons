---
title: Reset Number Input
---

A [preflight](https://tailwindcss.com/docs/preflight) that removes the arrows from the number inputs.

```css
/* NOTE: for some reason `@base` didn't work here. */
@layer base {
    input[type="number"] {
        @apply reset-number-input;
    }
}
```

:::note
Usage as a [utility](/utilities/reset-number-input).
:::
