---
title: Reset Search Input
---

A [preflight](https://tailwindcss.com/docs/preflight) that removes the x button from the search inputs.

```css
/* NOTE: for some reason `@base` didn't work here. */
@layer base {
    input[type="search"] {
        @apply reset-search-input;
    }
}
```

:::note
Usage as a [utility](/utilities/reset-search-input).
:::
